Collect all gems to open the exit. Use the arrow keys to move.

This game is the result of a long weekend spent learning TypeScript and the Phaser game framework, with the objective of getting a working game by the end of four days. It is inspired by games written on the BBC Microcomputer, in particular Ebony Castle. The game is pretty basic but some people might find it a tolerably entertaining diversion.
